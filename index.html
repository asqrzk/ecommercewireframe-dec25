Project Overview

The objective is to build a scalable, high-performance e-commerce web application specialized for selling perfumes. The frontend will be a consumer-facing PWA (Progressive Web App), while the backend administration will be powered by Next.js for SSR capabilities and SEO management.

Consumer App

  • Next.js PWA
  • Tailwind CSS (Mobile First)
  • OTP-based Login

Admin Platform

  • Next.js Dashboard
  • Inventory & Order Management
  • Analytics & Reporting
